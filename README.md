# 📝 To-Do List com Engenharia de Prompt

Projeto desenvolvido como parte do desafio **"IA no Desenvolvimento de Software"**, com foco na aplicação de Engenharia de Prompt durante o ciclo de desenvolvimento.

---

## 🎯 Objetivo

Criar uma aplicação de lista de tarefas (To-Do List) funcional utilizando HTML, CSS e JavaScript, aplicando conceitos de:
- Planejamento com IA
- Geração consciente de código
- Revisão e melhoria contínua

---

## 🚀 Funcionalidades

- ✅ Adicionar tarefas
- 🎯 Definir prioridade (Alta, Média, Baixa)
- 📋 Listar tarefas dinamicamente
- ✔️ Marcar tarefas como concluídas
- 🗑️ Excluir tarefas
- 📦 Histórico de tarefas concluídas
- 🔎 Filtros por prioridade e status

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ Como executar o projeto localmente

### 🔹 Método 1 (Simples)
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

---

### 🔹 Método 2 (Recomendado - VS Code)
1. Abra a pasta no VS Code
2. Instale a extensão **Live Server**
3. Clique com o botão direito no `index.html`
4. Selecione **"Open with Live Server"**

---
